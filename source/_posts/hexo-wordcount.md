只需要安装一个插件 WordCount

## 安装
```bash
$ npm i hexo-wordcount --save
```
## 使用
单篇文章字数
```html
<%=wordcount(post.content) %>
```
所有文章的总字数
```html
<%=totalcount(site) %>
```
